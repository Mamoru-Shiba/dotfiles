## dotfiles

## 初回
make

## Makefile
記法
```
<ターゲット>: <ソースファイル>
    [コマンド]
```
- ターゲット（生成物）
- ソースファイル（ターゲットを作る為に必要なファイル）
- コマンド （ターゲットを入力から作るためのコマンド）

## 動作
.bin配下のファイルのシンボリックリンクを作成
